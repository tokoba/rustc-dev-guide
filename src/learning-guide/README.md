# rustc-dev-guide 学習ガイド

この学習ガイドは、Rustコンパイラ（rustc）の開発に興味がある方々を対象とした体系的な学習リソースです。初級者から上級者まで、各自のレベルや目的に応じた学習パスを提供し、効果的な学習体験をサポートします。

## この学習ガイドについて

### 目的

- Rustコンパイラの開発への参加を希望する方々に体系的な学習経路を提供
- 既存の技術ドキュメントと実践的な学習を結びつける
- コミュニティへの貢献活動を促進する

### 特徴

- **段階的学習**: 初級者から上級者まで、無理なくステップアップできる構成
- **実践的アプローチ**: 理論学習だけでなく、実際の貢献活動に繋がる内容
- **柔軟な学習パス**: 目的や興味に応じて選択できる複数の学習経路
- **既存ドキュメントとの連携**: 詳細な技術情報は既存のrustc-dev-guideを参照

## 学習の進め方

### 学習の前提条件

- Rustの基本的な文法知識（所有権、ライフタイム、トレイトなど）
- Gitの基本的な使用方法
- コマンドラインの基本的な操作
- 開発環境（Rustツールチェーン）のセットアップ

### 効率的な学習方法

1. **自分のレベルを正しく評価**: 各学習パスの前提条件を確認
2. **理論と実践のバランス**: ドキュメントを読むだけでなく、実際に手を動かす
3. **小さな目標設定**: 大きな目標を小さな達成可能なタスクに分割
4. **コミュニティの活用**: 分からないことは積極的に質問し、議論に参加する

### コミュニティへの参加方法

- [Zulipチャット](https://rust-lang.zulipchat.com/#narrow/stream/182449-t-compiler.2Fhelp): 質問や議論
- [GitHub Issues](https://github.com/rust-lang/rust/issues): バグ報告や機能提案
- [RFCプロセス](https://github.com/rust-lang/rfcs): 新機能の設計議論

## 学習ガイドの使い方

### ナビゲーション

- 各ページには前後のページへのリンクが配置されています
- 関連トピックへの相互参照を活用して、横断的な学習を進められます
- 各学習パスのREADME.mdには、そのパスの詳細な学習計画が記載されています

- 各学習パスには実践的な演習やプロジェクトが含まれています
- 実際のrustcリポジトリでの貢献を想定した課題に取り組めます
- 学習した知識をすぐに実践に活かせるよう設計されています
